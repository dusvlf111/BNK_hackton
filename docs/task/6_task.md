# 케어페이 가디언 - 작업 목록 6: 사용자 대시보드 및 거래 내역 UI

**작업 범위**: 메인 대시보드, 거래 내역, 알림 센터

---

### 관련 파일
- `app/user/dashboard/page.tsx` - 사용자 메인 대시보드
- `app/user/transactions/page.tsx` - 거래 내역 페이지
- `components/DashboardAlert.tsx` - 대시보드 알림 컴포넌트
- `components/WeeklySummary.tsx` - 주간 요약 위젯
- `components/TransactionList.tsx` - 거래 내역 리스트
- `app/api/transactions/list/route.ts` - 거래 내역 조회 API
- `app/api/alerts/list/route.ts` - 알림 목록 조회 API
